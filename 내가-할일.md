### 1) 작품 자동 수집 (sync-works) — 기본값

**지금 화제작(TMDB 트렌딩·이번 주)** 기준으로 영화·드라마를 가져와 **works**에 넣습니다. (한국 OTT 인기순보다 최신 화제에 맞게 갱신됨)

- **호출 방법 (수동 1회)**  
  - URL: `GET https://본인도메인.com/api/cron/sync-works`  
  - Header: `X-Cron-Secret: (CRON_SECRET 값)` 또는 `Authorization: Bearer (CRON_SECRET 값)`  
  - 영화 30편 + TV 30편이 works에 추가/갱신됩니다.
- **필수:** Vercel(또는 .env.local)에 **TMDB_API_KEY**가 들어 있어야 합니다.
- **자동 실행:** Vercel Cron에 포함되어 있어, **매주 목요일 새벽 4시(KST)** 에 한 번 실행됩니다. (`vercel.json`에서 스케줄 변경 가능)

### 2) 수동 리스트(엑셀)로 작품 추가 (import-works)

필요할 때 **작품명 리스트**를 보내서 기존 1번 데이터 **위에 추가**만 하는 API입니다. 엑셀에서 작품명만 복사해 JSON으로 보내면 됩니다.

- **호출 방법**  
  - URL: `POST https://본인도메인.com/api/cron/import-works`  
  - Header: `Content-Type: application/json`, `X-Cron-Secret: (CRON_SECRET 값)`  
  - Body 예시: `{ "titles": [ "대홍수", "오징어 게임 2", "듄" ] }`  
  - TMDB 검색 후 첫 번째 결과를 works에 추가합니다. 이미 있는 작품(tmdb_id 동일)은 건너뜁니다.
- **응답:** `{ "added": 2, "skipped": 1, "total": 3, "errors": [] }`  
  - 검색 실패·중복은 `errors` 또는 `skipped`로 확인 가능.

### 3) 작품에 수동 유튜브 영상 등록 (관리자 전용)

작품 상세 페이지 **영상 섹션 상단**에 넣을 유튜브 영상을 **직접 URL/ID로** 지정할 수 있습니다. (나만 사용)

- **호출 방법**  
  - URL: `PATCH https://본인도메인.com/api/admin/works/{작품slug}/manual-videos`  
  - Header: `Content-Type: application/json`, `X-Cron-Secret: (CRON_SECRET 값)`  
  - Body 예시: `{ "youtube_ids": [ "dQw4w9WgXcQ", "https://www.youtube.com/watch?v=다른ID" ] }`  
  - URL 전체를 넣어도 ID만 추출해 저장합니다.
- **DB:** `works.manual_video_ids` (배열). 마이그레이션 `002_work_manual_videos_and_end_date.sql` 적용 필요.

### 작품 상세 레이아웃 (영상 중심) & 동적화 매핑

작품 상세 페이지 순서: **제목 → 한 줄 요약 → 메타 정보 박스 → 영상(예고편/리뷰) → 시청 플랫폼 버튼 → 반응·리뷰 요약 → 수익 영역**.

- **어디서 볼까:** 넷플릭스/티빙 등 플랫폼별 **컬러 버튼** (1~2개 이상).
- **수익 영역:** 1️⃣ 공식 굿즈(있을 경우 이미지+「공식 굿즈 보러가기」) 2️⃣ 제휴 링크(📀 원작 만화 🎧 OST 📚 세계관 책 등). 나중에 Elementor Pro / ACF 업그레이드 시: 제목→Post Title, 요약→ACF 한 줄 요약, 플랫폼→ACF 링크, 영상→ACF YouTube URL 로 동적화 가능.

### 불러온 작품에 유튜브 영상 가져오기 (sync-videos)

works에 들어간 작품들(기묘한 이야기, 오징어게임 등)에 대해 **예고편·쇼츠·리뷰** 영상을 YouTube에서 가져옵니다. **YouTube가 메인, TMDB 예고편은 보조.**

**수집 구조**
- **영상 소스 우선순위:** 1) YouTube 공식 채널(넷플릭스/티빙 등) 2) YouTube 일반(리뷰/쇼츠) 3) TMDB 예고편(보조)
- **1단계:** `[작품명] 예고편`, `[작품명] 쇼츠`, `[작품명] 리뷰` 각각 검색, `order=viewCount` (조회수 기준). 쇼츠는 `videoDuration=short` 적용.
- **2단계:** 채널이 Netflix / Netflix Korea / TVING 등 공식 채널이면 매칭 점수에 보너스(+0.3) 적용해 우선 수집.
- **3단계:** 제목·길이로 자동 분류 — "예고/trailer" → trailer, "short/#shorts" 또는 60초 이하 → shorts, "리뷰/해설" → review.
- **4단계:** `youtube_id` unique로 DB 저장, 중복 방지. 티빙은 TMDB에 거의 없으므로 반드시 YouTube API + 채널 필터로 수집.

- **수동 1회 호출 (지금 하기)**  
  - URL: `GET https://ottsak.com/api/cron/sync-videos`  
  - Header: **X-Cron-Secret** = (Vercel에 넣은 CRON_SECRET 값)  
  - Postman 등으로 호출하면 **최근 30일 이내 등록된 works** 기준으로 YouTube 검색 후 `videos` 테이블에 저장됩니다.
- **자동:** Vercel Cron이 **매일 새벽 3시(KST)** 에 위 주소를 호출하므로, 그 이후엔 새로 넣은 작품도 자동으로 영상이 붙습니다.
