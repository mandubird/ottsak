# 옽싹 수익 구조 — 1차 모델 (제휴·굿즈)

> 영상 아래 공식 굿즈 / 제휴 링크는 **넣어야 하고**, MVP 단계부터 **조심스럽게, 구조적으로** 넣는 게 맞다.

---

## 1. 이 모델이 옽싹에 잘 맞는 이유

### ① 시청 직후 = 구매 욕구 최고

**사람 흐름:** 영상 본다 → 감정 올라옴 → 더 알고 싶음 / 소유하고 싶음  

👉 이 타이밍에 **굿즈, OST, 원작 도서, 쿠팡 링크** 노출 → **전환률 가장 높음**

### ② 광고처럼 안 보임

- 배너 광고 ❌  
- 영상 아래 **“관련 아이템”** ✅  

사람 인식: *“아, 이거 관련해서 살 만한 게 있구나”* → **UX 깨지지 않음**

### ③ 소규모 트래픽에서도 돈이 됨

- CPM 광고 → 트래픽 많이 필요  
- 제휴 링크 → **소수 클릭만 있어도 수익 가능**  
👉 **MVP 단계에 최적**

---

## 2. “아무렇게나 넣으면” 망한다

❌ **이런 방식은 안 됨**

- 영상 바로 아래 쿠팡 배너
- 광고 문구 남발
- 모든 작품에 동일 링크

👉 **신뢰도 하락, SEO에도 안 좋음**

---

## 3. 옽싹식 “정석 배치 구조” (추천)

### 작품 상세 페이지 흐름

```
[작품 정보]
[한 줄 요약]
[예고편 / 리뷰 영상]   ← 사람들이 반응하는 지점

        ↓

[관련 아이템]
- 공식 굿즈
- OST
- 원작 도서
(※ ‘제휴 링크 포함’ 소형 표기)

[사람들 코멘트]  (확장 시)
```

👉 **광고가 아니라 콘텐츠 연장선**으로 느껴지게.

---

## 4. 어떤 링크가 가장 잘 맞냐

### 1순위 (강력 추천)

- **공식 굿즈** (넷플릭스 스토어 등)
- **OST** (쿠팡 / 애플뮤직 / 멜론 링크)
- **원작 웹툰 / 소설**

### 2순위 (조건부)

- 관련 게임
- DVD / 블루레이
- 피규어 (인기 IP만)

---

## 5. 구현 시 체크리스트 (MVP)

- [ ] 작품 상세에서 **영상 블록 아래**에만 **[관련 아이템]** 섹션 배치
- [ ] 항목별: 공식 굿즈 / OST / 원작 도서 등 **작품별로 다른 링크** (동일 링크 반복 X)
- [ ] **“제휴 링크 포함”** 소형 표기 한 곳에만
- [ ] DB/관리: 작품별 “관련 링크” 필드 또는 테이블 (선택)

---

*이 문서는 수익 구조 1차 모델 정리용이며, 실제 링크/제휴 정책은 운영 정책에 따름.*
